## 🔧 Troubleshooting
If you can't connect:
1. Make sure your phone and computer are on the same network
2. Try refreshing the Expo Go app
3. Check that the development server is still running

The mobile app connects to the same backend as your web app, so all your data and users are shared between platforms!
